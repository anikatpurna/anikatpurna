## Featured Project

### Customer Review Analysis: Rating vs. Sentiment
An end-to-end data science project using 23,000+ women's clothing reviews
to explore whether customer sentiment matches their star ratings —
and what happens when it doesn't.

- Exploratory data analysis across rating, sentiment, and review length
- VADER-based sentiment scoring to quantify emotional tone in text
- Decision tree classifier to predict recommendation behavior
- Interactive Rating vs. Sentiment dashboard with CSV export
